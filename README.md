# gitleak
在lijiejie大佬的GitHack（https://github.com/lijiejie/GitHack）基础上修改的，添加了通过文件名下载的功能。

使用方法：
python gitleak.py http://www.target.com/.git/  

如下：使用GitHack原工具时，提示File not found。
